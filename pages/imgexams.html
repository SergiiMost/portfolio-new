IMGexams

Description

IMGexams is a web platform for international medical graduates who are preparing for exams in order to qualify for medical residency matches in Canada and the USA. Currently, there is no centralized place where medical graduates can discuss preparation for Canadian exams (MCCQE Part I, MCCQE Part II , NAC ) , share useful tips, answer questions, etc. There is a lack of easily accessible information regarding exams mentioned before,and IMGexams is being developed to solve this issue.

At this point, it’s a minimum viable product (MVP), and it’s in a beta testing stage. It is in an active development phase and may change significantly based on feedback. Some very useful features are going to be implemented after the current testing iteration and adjustments ( according to the feedback ).

Conclusion

I've learned that a project implementation should not start before all aspects of it are clearly understood. Also, it's very important to plan ahead how everything is structured in your application. Diagrams can be very helpful in this case. Plan 7 times, code once!
